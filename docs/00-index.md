## 关键设计

1. **观测优先**：所有数据来自系统命令，不修改任何系统状态
2. **教学优先**：每一个数据点都带"它是什么 + 为什么"的解释
3. **零持久化**：不保存历史，只展示此刻
4. **本地优先**：数据不出本机
